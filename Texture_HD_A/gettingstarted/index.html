Pour démarrer

Ce dossier réunit les cartes de référence qui accompagnent chaque texture de la collection. Pour une même image source, on retrouve la couleur (base), la version tileable (t), le bump (b), la normale (n), le spéculaire (s), le réflectif (r) et le mate (m). Cliquez sur une vignette pour l'agrandir, Échap pour fermer.

Suffixes utilisés

  • (rien) — texture de base, couleur diffuse.
  • t — version tileable, sans coutures visibles.
  • b — bump map, niveaux de gris pour relief simulé.
  • n — normal map, encode les normales en RVB.
  • s — specular map, contrôle la brillance.
  • r — reflection map, pour les réflexions environnementales.
  • m — masque de découpe ou matte.
